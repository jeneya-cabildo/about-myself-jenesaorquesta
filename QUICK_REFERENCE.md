## 📸 Screenshots - How to Capture

### Light Mode
1. Open http://localhost:3000
2. Click Moon icon (top right)
3. F12 → More tools → Screenshot
4. Save as `light-mode.png`

### Dark Mode
1. Click Sun icon (top right)
2. F12 → More tools → Screenshot
3. Save as `dark-mode.png`

### Mobile
1. F12 → Toggle device (Ctrl+Shift+M)
2. Select iPhone 12 Pro
3. Repeat light/dark mode capture

## ❓ FAQ

**Q: Where can I see the live website?**

**Q: How do I test dark/light mode?**
A: Click the Sun/Moon icon in the top-right navigation bar.

**Q: How often does it deploy?**
A: Every time you push to GitHub, Vercel automatically deploys within 1-2 minutes.

**Q: Can I make changes?**
A: Yes! Edit any component, push to GitHub, and Vercel will auto-deploy.

**Q: Is there a backend?**

**Q: How is SEO handled?**
