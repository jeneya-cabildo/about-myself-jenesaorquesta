# Jenesa C. Orquesta - Professional CV Website

A modern, responsive CV website built with **Next.js 15**, **Tailwind CSS**, and **Shadcn UI**, showcasing a Full Stack and Agentic AI Developer's professional profile.

## 🎯 Project Overview

This is a professional CV website demonstrating modern web development practices and AI-powered interface generation using v0.dev. The site features a comprehensive profile, skills showcase, experience timeline, education background, and contact information.

**Live Demo**: [Deployed on Vercel](https://about-myself-jenesaorquesta.vercel.app)
**GitHub Repository**: [jeneya-cabildo/about-myself-jenesaorquesta](https://github.com/jeneya-cabildo/about-myself-jenesaorquesta)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/jeneya-cabildo/about-myself-jenesaorquesta.git
cd about-myself-jenesaorquesta

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 🐛 Troubleshooting

### Build Issues
```bash
# Clear Next.js cache
rm -r .next

# Reinstall dependencies
rm -r node_modules package-lock.json
npm install
```

### Theme Not Persisting
- Check localStorage is enabled in browser
- Clear browser cache and cookies
- Try incognito/private browsing

### Performance Issues
- Check Network tab for large assets
- Run `npm run build` and `npm start` for production-like environment
- Use Chrome DevTools Performance tab for analysis
